

	관리자(root) 계정의 암호 분실시 변경 방법(mysql이 path로 설정되어 있는 경우 관리자 권한으로 cmd를 실행)

	1) mysqladmin -u root -p '새비밀번호'

	2) mysql -u root mysql
		update user set password=password('새비밀번호') where user='root';
		flush privileges;

	-- 1), 2)는 기존 비밀번호가 설정되어 있지 않은 경우 설정하는 방법

	3) set password for root=password('새비밀번호')
